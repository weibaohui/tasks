# 文档目录

## 规范 (standards/)

开发必须遵守的约束规则，包含：
- 文档编写规范
- 技术架构规范
- Git 工作流规范
- 测试规范

## 指南 (guides/)

开发过程中的操作参考文档，包含：
- 问题排查指南
- 数据库指南
- 部署指南

---

## 文件命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 规范文件 | `序号-名称.md` | `01-document.md` |
| 指南文件 | `名称.md` | `debugging.md` |

## 编号规则

- `01` - 基础规范（文档、架构）
- `02` - 架构规范
- `03` - 流程规范
- `04` - 测试规范
- `10+` - 专项规范
